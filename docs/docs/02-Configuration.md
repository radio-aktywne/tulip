## Environment variables

You can configure the app at runtime using various environment variables:

- `TULIP__APIS__BEAVER__HOST` -
  host of the beaver API
  (default: `localhost`)
- `TULIP__APIS__BEAVER__PATH` -
  path of the beaver API
  (default: ``)
- `TULIP__APIS__BEAVER__PORT` -
  port of the beaver API
  (default: `10500`)
- `TULIP__APIS__BEAVER__SCHEME` -
  scheme of the beaver API
  (default: `http`)
- `TULIP__APIS__ICANHAZDADJOKE__HOST` -
  host of the icanhazdadjoke API
  (default: `icanhazdadjoke.com`)
- `TULIP__APIS__ICANHAZDADJOKE__PATH` -
  path of the icanhazdadjoke API
  (default: ``)
- `TULIP__APIS__ICANHAZDADJOKE__PORT` -
  port of the icanhazdadjoke API
  (default: ``)
- `TULIP__APIS__ICANHAZDADJOKE__SCHEME` -
  scheme of the icanhazdadjoke API
  (default: `https`)
- `TULIP__DEBUG` -
  enable debug mode
  (default: `true`)
- `TULIP__SERVER__HOST` -
  host to run the server on
  (default: `0.0.0.0`)
- `TULIP__SERVER__PORT` -
  port to run the server on
  (default: `10530`)
